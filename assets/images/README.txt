NOTES FOR BUYER
---------------
- Minimum resolution: 2x the display size for retina screens (e.g. hero needs 1920px wide min)
- Format: JPEG for photos (quality 80-85%), PNG only for logos/graphics with transparency
- File size: compress before upload — hero < 400KB, cards < 120KB, gallery < 200KB
- Alt text: update every alt="" attribute in the HTML to describe the actual photo content
- Hosting: place images in this /assets/images/ folder OR update src paths to your CDN
